# 💼 FinTrack Pro

Aplikasi pencatat keuangan & portofolio investasi untuk freelancer dan investor.

## ✨ Fitur
- 💰 Catat pemasukan & pengeluaran tak terbatas
- 📊 Lacak portofolio (Saham, Kripto, Reksa Dana, Emas, Obligasi)
- 📈 Simulasi harga live portofolio
- 🎨 Tema warna bisa dikustomisasi
- 📱 Mobile-first, bisa di-install sebagai PWA
- 💾 Data tersimpan otomatis di browser (localStorage)

## 🚀 Cara Jalankan

### Install & Dev
```bash
npm install
npm run dev
```
Buka http://localhost:5173

### Build Production
```bash
npm run build
```

### Preview Build
```bash
npm run preview
```

## 🌐 Deploy ke Vercel
```bash
npm install -g vercel
vercel
```
Atau drag & drop folder `dist` ke vercel.com

## 📱 Install sebagai PWA
Setelah deploy, buka di Chrome HP → menu → "Add to Home Screen"

## 🛠️ Tech Stack
- React 18
- Vite
- Tailwind CSS
- Recharts
- Lucide React
